## Nombre del proyecto

*Generador de Contraseñas Seguras con Sonido*

---

## Objetivo del sistema

Desarrollar un sistema en Python que permita generar contraseñas seguras de forma automática, evaluando su nivel de seguridad y mostrando una alerta sonora o visual según el resultado obtenido.

---

## Descripción del proyecto

CipherSound es un programa desarrollado en Python que permite al usuario crear contraseñas aleatorias de acuerdo con las opciones seleccionadas. El sistema solicita la longitud de la contraseña y permite incluir números, letras mayúsculas y símbolos.

Luego, el programa evalúa si la contraseña generada es fuerte o débil. Además, incluye una alerta sonora o visual para indicar el resultado de seguridad, mejorando la interacción entre el usuario y el sistema.

## Contexto del Proyecto

El proyecto CipherSound fue pensado como una herramienta básica orientada a fortalecer la seguridad digital en entornos donde se maneja información sensible esto puede ser en oficinas o instituciones que trabajan con datos importantes, el uso de contraseñas débiles puede representar un riesgo para la protección de la información.

es por esto que el sistema permite generar contraseñas más seguras mediante la combinación de letras, números, mayúsculas y símbolos, promoviendo mejores prácticas en el manejo de accesos digitales.
---

## Funcionalidades del sistema

- Generar contraseñas aleatorias.
- Solicitar la longitud de la contraseña.
- Permitir incluir números.
- Permitir incluir letras mayúsculas.
- Permitir incluir símbolos.
- Evaluar el nivel de seguridad de la contraseña.
- Mostrar si la contraseña es fuerte o débil.
- Activar una alerta sonora o visual.
- Permitir generar otra contraseña si el usuario lo desea.

## Relación con la asignatura
El sistema CipherSound puede ejecutarse en computadora con Windows utilizando Python instalado, Visual Studio Code o la consola de sistema.
Tambien puede ejecutarse en compiladores online compatibles con Python, tambien la alerta sonora puede variar segun la plataforma. Es posible que en algunos compiladores online el sonido no se reproduzca, sin embargo enviara un mensaje visual indicando si la contraseña generdo es fuerte o debil.
El programa utiliza print("\a") como alerta sonora básica de consola.
